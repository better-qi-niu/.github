# 简介

**奇牛编程是一家专注于IT职业教育的在线教育企业。 他们的目的是让所有IT学习者，不分年龄、不分职业，平等地享有一流的IT教育服务。**

# 关于开源组织创建者

本开源组织的创建者[AngryKeyboard](https://gitee.com/AngryKeyboard)曾以 **“名侦探毛利小五郎”** 的身份活跃于奇牛学院，随后被宣传为 **“明星学员”** 。先后两次以 **“牛肉”** 、 **“牛盛”** 身份担任助教，是奇牛学院头号铁粉。
